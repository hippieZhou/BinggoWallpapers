## 产品功能

> 产品主要功能的简短摘要; 显示为项目符号列表; 最多 20 个应用功能

**中文：**
- 支持 14 个地区的必应壁纸（中国、美国、英国、日本、德国、法国、西班牙、意大利、俄罗斯、韩国、巴西、澳大利亚、加拿大、印度）
- 实时浏览最新的必应每日壁纸
- 浏览历史壁纸归档，支持增量加载
- 多分辨率下载（1366x768、1920x1080、1920x1200、3840x2160 4K）
- 7 种图像特效（曝光、色温、色调、模糊、对比度、饱和度、像素化）
- 实时特效预览和编辑
- 高质量导出（PNG/JPEG，支持 2x 高分辨率）
- 设备 Mockup 预览功能
- 本地 SQLite 数据库存储，支持离线浏览
- 现代化 WinUI 3 Fluent Design 界面
- 响应式布局，适配不同屏幕尺寸
- 流畅的动画过渡效果
- 优雅的加载状态提示
- 友好的错误处理和重试机制
- 应用内通知和系统通知支持

**English:**
- Support for Bing wallpapers from 14 regions (China, USA, UK, Japan, Germany, France, Spain, Italy, Russia, South Korea, Brazil, Australia, Canada, India)
- Browse the latest Bing daily wallpapers in real-time
- Browse historical wallpaper archives with incremental loading
- Multi-resolution downloads (1366x768, 1920x1080, 1920x1200, 3840x2160 4K)
- 7 image effects (Exposure, Temperature, Tint, Blur, Contrast, Saturation, Pixelate)
- Real-time effect preview and editing
- High-quality export (PNG/JPEG, supports 2x high resolution)
- Device mockup preview feature
- Local SQLite database storage with offline browsing support
- Modern WinUI 3 Fluent Design interface
- Responsive layout adapting to different screen sizes
- Smooth animation transitions
- Elegant loading state indicators
- Friendly error handling and retry mechanisms
- In-app and system notification support
